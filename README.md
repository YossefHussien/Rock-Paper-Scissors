🪨📄✂️ Rock Paper Scissors

A console-based Rock Paper Scissors game built with C++ as part of my programming learning journey.


🎮 Features


🎲 Random computer moves
🔢 Choose number of rounds (1–10)
🏆 Score tracking per round and overall
🎨 Color-coded results (green = win, red = lose, yellow = draw)
🔔 Sound alert on computer win
🔁 Play again option after each game



🛠️ Built With


Language: C++
Concepts used: Enums, Structs, Functions, Loops, Switch statements, Randomization



🚀 How to Run


Clone the repository:


bash   git clone https://github.com/YossefHussien/Rock-Paper-Scissors.git


Open the project in your IDE (Visual Studio recommended)
Build and run Rock-Paper-Scissors.cpp
